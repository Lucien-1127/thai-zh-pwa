# 泰中雙語 PWA — ไทย-จีน PWA

這是一個**泰文／中文雙語**漸進式網頁應用程式（Progressive Web App, PWA），支援離線使用、可安裝至主畫面，提供流暢的手機與桌面瀏覽體驗。

## 功能特色

- 🇹🇭🇹🇼 泰中雙語介面
- 📱 PWA 支援 — 可加入主畫面，離線亦可使用
- ⚡ Service Worker 快取加速載入
- 🎨 響應式設計，手機與桌面皆適用

## 使用方式

直接在瀏覽器中開啟 `index.html` 或透過 GitHub Pages 存取。

## 開發

本專案為純靜態前端應用，無需後端伺服器。修改 `index.html`、`service-worker.js` 或 `manifest.json` 即可。

```bash
# 開啟本機開發伺服器（需有 Python）
python3 -m http.server 8000
# 或使用 Node.js
npx serve .
```
